Color Scheme Reference

Proper surface hierarchy to avoid the "everything is white/black" problem

The Problem vs. The Solution
Your Current Issue

Everything is the same shade. Sidebar, cards, and background all use similar colors like #1a1a1a vs #1f1f1f. No visual hierarchy. Elements blend together.

The Fix

Use 3 distinct surface levels: Canvas #09090b, Surface #18181b, and Borders #27272a. Cards get borders. Clear separation.

Copy-Paste CSS Variables
/* ============================================
   LIGHT MODE (Default)
   ============================================ */
:root {
  /* Backgrounds */
  --bg-canvas: #FAFAFA;     /* Page background */
  --bg-surface: #FFFFFF;    /* Cards, sidebar, panels */
  --bg-elevated: #FFFFFF;   /* Modals, dropdowns (+ shadow) */
  --bg-hover: #F5F5F5;      /* Hover states */

  /* Borders */
  --border: #E5E5E5;        /* Default borders */
  --border-subtle: #F0F0F0; /* Dividers, table rows */

  /* Text */
  --text-primary: #171717;   /* Headlines, important */
  --text-secondary: #525252; /* Body text */
  --text-muted: #A3A3A3;    /* Labels, hints */

  /* Accents */
  --accent-primary: #2563EB;  /* Buttons, links */
  --accent-secondary: #7C3AED; /* Schedules */
  --success: #059669;       /* Online, success */
  --warning: #D97706;       /* Pending, drafts */
  --danger: #DC2626;        /* Offline, errors */

  /* Shadows (light mode only) */
  --shadow-sm: 0 1px 2px rgba(0,0,0,0.05);
  --shadow-md: 0 4px 6px -1px rgba(0,0,0,0.1);
  --shadow-lg: 0 10px 15px -3px rgba(0,0,0,0.1);
}

/* ============================================
   DARK MODE
   ============================================ */
.dark {
  /* Backgrounds */
  --bg-canvas: #09090B;     /* Near black, NOT gray */
  --bg-surface: #18181B;    /* Cards, sidebar */
  --bg-elevated: #27272A;   /* Modals, dropdowns */
  --bg-hover: #27272A;      /* Hover states */

  /* Borders (MORE IMPORTANT in dark mode) */
  --border: #27272A;        /* Default borders */
  --border-subtle: #3F3F46; /* Dividers */

  /* Text */
  --text-primary: #FAFAFA;   /* Headlines */
  --text-secondary: #A1A1AA; /* Body text */
  --text-muted: #71717A;    /* Labels */

  /* Shadows (disable in dark mode) */
  --shadow-sm: none;
  --shadow-md: none;
  --shadow-lg: none;
}
3 Rules to Fix Your UI
Rule 1

Canvas ≠ Surface ≠ Elevated

Use 3 distinct background colors. The page background should be different from card backgrounds. Cards should have borders in dark mode.

Rule 2

Every Card Gets a Border

In dark mode, shadows don't work. Cards MUST have a 1px solid var(--border) to be visible. This is non-negotiable.

Rule 3

Blue Primary, Not Orange

Orange/amber = warning. Blue = action. Use #2563EB for buttons and links. Reserve orange for "pending" or "draft" states only.
